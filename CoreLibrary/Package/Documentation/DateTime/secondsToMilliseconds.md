# `secondsToMilliseconds` - Convert amount of seconds to amount of milliseconds

Converts amount of seconds to amount of milliseconds.

```
(secondsAmount: number): number
```


## Examples

```typescript
console.log(0.005); // -> 5
console.log(0.05); // -> 50
console.log(0.5); // -> 500
console.log(1); // -> 1000
console.log(2); // -> 2000 
```

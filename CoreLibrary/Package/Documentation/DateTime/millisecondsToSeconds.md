# `millisecondsToSeconds` - convert amount of milliseconds to amount of seconds

Convert amount of milliseconds to amount of seconds.

```
(millisecondsAmount: number): number
```


## Examples

```typescript
console.log(5); // -> 0.005
console.log(50); // -> 0.05
console.log(500); // -> 0.5
console.log(1000); // -> 1
console.log(2000); // -> 2
```
